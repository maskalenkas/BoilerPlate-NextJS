# Tecnologias nesse boilerplate:

- Husky
- Lint-Staged

- NextJS ( yarn start )
- Jest ( yarn test, yarn test:watch )

- DependaBot ja configurado para não quebrar
- PWD
- StoryBook ( yarn storybook )

- componente mediaWatch para escolher renderizar ou não algo
